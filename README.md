# 山居秋暝：青绿江山数字手卷

一个以王维《山居秋暝》为核心的数字人文专题网页。项目把诗歌四联的推进转译为横向手卷路径，并通过文本细读、作者内部比较、接受史和策展说明，回答：一座看似“空”的秋山，为什么最终成为“王孙自可留”的生活世界？

## 直接运行

打开 `index.html` 进入英文主页面；打开 `zh.html` 进入中文页面。两页均可直接双击运行，不需要依赖、构建工具或服务器。

## 核心功能

- 入山启卷动画
- 鼠标拖动、滚轮、键盘和连续进度条浏览长卷
- 六段按钮准确跳转到雨、月、泉、竹、莲舟与“可留”场景
- “读诗句／看路径／查依据”三层解释
- 首联、颔联、颈联、尾联语法地图
- 《招隐士》《鹿柴》《竹里馆》和苏轼题跋文本对话
- 蓝铜矿真实标本与青绿概念分层
- A／B／C／D 证据分级和研究边界
- 响应式移动布局与键盘可访问性

## 数字人文方法

页面统一区分：

- **A 一手文本**
- **B 文本分析／通行阐释**
- **C 接受史**
- **D 策展转译**

《雨霁秋山》是本项目依据诗句顺序绘制的原创 SVG 数字手卷，不是古画，也不是王维实景复原。蓝铜矿标本只用于解释网页的青绿色彩媒介。英文译文同样为项目自译，并对 `empty`、`downriver` 和 `Wangsun` 等解释边界作出说明。

## GitHub Pages

项目已配置 GitHub Actions。将代码推送至公开 GitHub 仓库的 `main` 分支，并在 **Settings → Pages** 中选择 **GitHub Actions**，即可获得公开链接。详见 [DEPLOYMENT.md](DEPLOYMENT.md)。
