THE CORE INTERACTION

One poem,
six precise stops.

Drag freely, use the range control, or select a line below. Each stop keeps the Chinese text, project translation and matching image in one view.

空山新雨后天气晚来秋

After fresh rain in the empty mountains,
the evening air turns autumnal.

明月松间照

The bright moon shines
between the pines.

清泉石上流

A clear spring flows
over stones.

竹喧归浣女

Bamboo resounds—
the washerwomen return.

莲动下渔舟

Lotus leaves stir—
a fishing boat moves downriver.

随意春芳歇王孙自可留

Let spring blossoms fade:
Wangsun—you may remain.

A SMALL, EXPLAINABLE STACK

Four front-end ideas
build the page.

The project deliberately avoids frameworks. Each visible effect can be explained from a short piece of HTML, CSS or JavaScript.

01 · HTML

Structure the meaning

Semantic sections, articles, language attributes and accessible buttons keep poem, translation and controls distinct.

<article class="scene">
02 · CSS

Compose the handscroll

Grid builds the museum frame; Flexbox lays six scenes in one horizontal sequence; media queries reshape it on mobile.

display: flex;
03 · MOTION

Animate one state

A class change opens the scroll. CSS controls the clip-path transition, so motion remains purposeful and easy to explain.

.hero.opened { … }
04 · JAVASCRIPT

Synchronise the view

One current-stop value updates position, note, range and active button. No framework or chart library is required.

scrollTo({ left })
